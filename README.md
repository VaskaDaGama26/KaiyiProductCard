# Проект: Kaiyi X3 Pro Landing Page

Этот проект представляет собой лендинг для автомобиля **Kaiyi X3 Pro**, реализованный на чистом **HTML**, **CSS** и **JavaScript**.

Форматирование кода поддерживается с помощью **Prettier**.

---

## Стек технологий

- HTML5
- CSS3 (с использованием CSS-переменных и адаптивной верстки)
- JavaScript (Vanilla)
- Prettier (для автоматического форматирования кода)

---

## Установка и запуск

1. **Скачайте или клонируйте проект**  
   git clone <https://github.com/VaskaDaGama26/KaiyiProductCard>
2. **Откройте проект в VS Code**
3. **Запустите локальный сервер для просмотра страницы**
   - Используйте расширение Live Server
     для VS Code: - Правой кнопкой мыши по index.html → Open with Live Server

Либо воспользуйтесь встроенными средствами других редакторов или локального веб-сервера.
